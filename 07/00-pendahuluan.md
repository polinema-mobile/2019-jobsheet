# Pendahuluan

Pada praktikum kali ini anda akan mempelajari lebih dalam mengenai Android UI.

## Tujuan Pembelajaran

- Mahasiswa mampu membuat splash screen.
- Mahasiswa mampu membuat view pager.
- Mahasiswa mampu membuat bottom navigation bar.

2. Android Studio

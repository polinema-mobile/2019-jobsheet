# Developer Options

> Tampilan menu mungkin akan berbeda tergantung vendor mobile phone.

- Buka menu **Settings**
- Pilih menu **About phone**
- Cari menu **Build number**, tekan 7 kali hingga keluar popup **You are now a
 developer!**
